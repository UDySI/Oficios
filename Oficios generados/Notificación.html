¿Qué sigue?

  • Abre el borrador para revisar, comentar y modificar el oficio.
  • Abre el tablero Oficios para ver la lista de oficios creados y para cambiar el status del documento:
    • Cuando el archivo esté listo para ser firmado, cambia el status a FINAL para:
      • retirar la marca de agua,
      • actualizar los datos del oficio en la base de datos,
      • permitir el acceso para cualquier persona con el link y
      • habilitar el link para descargar el PDF.
    • Si el documento no será remitido por cualquier razón, cambia el status a CANCELADO.
      • El documento ya no será accesible desde el tablero.
      • El número de oficio consecutivo no será reasignado a oficios futuros.
